### 4. What is the output of the following program?

```c
#include <stdio.h>

int main (void) 
{
   int rows = 5, number = 1;
   
   for (int i = 0; i <= rows; i++) 
   {
      for (int j = 0; j <= i; j++) 
      {
         printf("%d ", number);
         number++;
      }
      printf("\n");
   }
}
 ```
